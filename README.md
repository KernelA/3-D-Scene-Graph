# 3D-Scene-Graph
3D scene graph generator from RGBD video, based on [FactorizableNet](https://github.com/yikang-li/FactorizableNet), implemented in Pytorch.

* Ubuntu 16.04
* Python 2.7

```

cd pcl-1.7.2

sudo dpkg -i pcl_*.deb
```
